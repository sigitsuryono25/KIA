Bagaimana cara menginput/mengubah data pada Imunisasi

Cara ini berlaku untuk setiap tahapan imunusasi baik Imunsasi 0-12, Diatas 1 Tahun, Bulan Imunisasi Anak Sekolah (BIAS), Imunisasi Tambahan serta Imunisasi Vaksin Lain (Imunisasi Lain).

PENTING !!!
Setiap Perubahan yang dilakukan, harus selalu mengklik tombol save. Bila tidak, perubahan yang telah dibuat tidak akan mengubah data apapun atau akan terjadi perubahan data yang tidak sesuai dengan keinginan.

Menginput Data

  1. Admin dapat memilih nama anak terlebih dahulu sebelum mulai menginputkan tanggal imunisasi ataupun catatan imunisasi.
  2. Tanggal imunisasi diisikan dengan cara mengklik kolom yang akan diisikan oleh tanggal imunisasi. Catatan imunisasi diisikan dengan cara mengklik ikon 'tangan'
  3. Sesudah diklik kolom yang akan diubah, akan muncul pilihan untuk memasukkan tanggal atau menghapus tanggal. Pilih Sesuai kebutuhan.
  4. Bila memilih memasukkan tanggal maka akan muncul tampilan untuk memilih tanggal. Tanggal terpilih akan langsung tercatat pada kolom yang dipilih tadi.
  5. Admin dapat memasukkan catatan terlebih dahulu jika dirasa perlu dengan mengklik ikon 'tangan' yang ada pada kanan atas dari tampilan dan dapat memasukkan catatan. Jika dirasa sudah sesuai dan tidak ada tambahan dapat langsung klik tombol save untuk mengirimkan data ke server.
  6. Jika memilih menghapus tanggal kolom akan menjadi kosong. Setelah itu klik tombol save untuk menyimpan perubahan ke server.

  4. Bila memilih memasukkan tanggal maka akan muncul tampilan untuk memilih tanggal. Tanggal terpilih akan langsung tercatat pada kolom yang dipilih tadi.
  5. Admin dapat memasukkan catatan terlebih dahulu jika dirasa perlu dengan mengklik ikon 'tangan' yang ada pada kanan atas dari tampilan dan dapat memasukkan catatan. Jika dirasa sudah sesuai dan tidak ada tambahan dapat langsung klik tombol save untuk mengirimkan data ke server.
